ABOUT ME

Hey Guys, I'm Jovan, and I am currently training to become a Cloud Engineer. I have Cloud experience in both AWS and Azure. CI/CD experience with GitHut Actions, Jenkins and Azure Devops Services. My programming language of choice has become C#, while learning .NET as a developing platfrom and Azure Funtions. I've picked up Infrastructure as Code via Terraform, provisioning resources on both AWS and Azure. Also carry experience with Containers using Docker, as well as Orchestrations with Kubernetes using EKS and AKS. My newly launched blog, "Up in the Clouds..." is managed on AWS Lightsail with a a custom domain name, content distribution and security with Cloudflare. The email address on both this website, and my blog were created with Amazon WorkMail, pointing to the same DNS as the blog.

I built this page with the guidance of the Cloud Guru Azure Resume Challenge. This was a great way to pull together a lot of services I've read about, or labbed with. This website is built on Azure, with a Static Site via Blob Storage, a CosmosDB that uses Azure Funtions to count the visitors (HTTP requests) and display with simple Javascript code. My twist was using a Cloudflare DNS, with managed SSL Certification and CDN. This websites also set up with a CI/CD pipeline with GitHub Actions. The front end updates the Blob index.html, and then Purges the Cache in Cloudflare for immediate updating. Any backend revision is tested before deployment to ensure the math calculates properly.
